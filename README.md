
## Introduction

`lasvdgp` is an R package of local approximate SVD-based GP
model, which is intended to solve the big O problem of the Gaussian
process model fitting for computer experiments with time series output
and large scale input.

## Installation

The installation follows the typical way of R packages on Github:

```r
library(devtools)
install_github("heavenmarshal/lasvdgp")
```
